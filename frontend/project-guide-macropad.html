MonkiiPad Macro Build Guide

Build your own programmable 12-key macropad for productivity

Prerequisites

Time Required

1-2 hours

Skill Level

Beginner

Tools Needed

Screwdriver, Soldering iron (optional)

What You'll Need

1

MonkiiPad PCB

12-key hot-swap PCB with rotary encoder support

2

Acrylic Case Layers

Laser-cut acrylic case pieces

3

Mechanical Switches (12x)

Cherry MX compatible switches

4

1u Keycaps (12x)

Standard 1u profile keycaps

5

Rotary Encoder (Optional)

EC11 rotary encoder with knob

6

Hardware Kit

M2 screws, standoffs, and USB-C cable

Build Steps

Step 1

Prepare Case Layers

Unpack and prepare all acrylic case layers for assembly.

  • Remove protective film from all acrylic pieces
  • Identify bottom, middle, and top layers
  • Clean any residue with microfiber cloth
  • Organize M2 screws and standoffs
üí° Pro Tip: Keep the protective film on until final assembly to avoid scratches!
Step 2

Install Bottom Standoffs

Attach M2 standoffs to the bottom acrylic layer.

  • Place bottom layer on flat surface
  • Insert M2 screws through mounting holes from bottom
  • Thread standoffs onto screws from top side
  • Tighten securely but don't over-tighten acrylic
Step 3

Install Rotary Encoder (Optional)

If using a rotary encoder, solder it to the PCB now.

  • Insert encoder legs through PCB holes
  • Ensure encoder is flush with PCB
  • Solder all 5 pins from the bottom
  • Clip excess pin length
  • Test encoder rotation before continuing
‚ö†Ô∏è Warning: Use proper ventilation when soldering and avoid breathing fumes!
Step 4

Insert Switches

Install your 12 mechanical switches into the hot-swap sockets.

  • Check all switch pins are straight
  • Align switch with hot-swap socket
  • Press down firmly until switch clicks in
  • Repeat for all 12 positions
  • Verify all switches are fully seated
Step 5

Stack Case Layers

Assemble the acrylic case by stacking all layers.

  • Place middle layers onto standoffs
  • Carefully lower PCB onto middle layers
  • Add top acrylic layer
  • Insert screws through top and tighten into standoffs
  • Ensure USB port is accessible
üí° Pro Tip: Tighten screws in a star pattern for even pressure distribution!
Step 6

Program Firmware

Flash QMK firmware and configure your macros.

  • Connect macropad via USB-C
  • Flash MonkiiPad firmware using QMK Toolbox
  • Open VIA configurator
  • Program each key with desired macro or function
  • Create multiple layers for different workflows
  • Configure rotary encoder if installed
üí° Pro Tip: Use layers to have 24, 36, or even 48 different macros accessible!
Step 7

Install Keycaps

Add keycaps and optional encoder knob to complete your build.

  • Press keycaps onto all 12 switches
  • If using encoder, attach the knob
  • Test all keys and macros
  • Customize RGB lighting if desired
