MonkiiBoard 30 Build Guide

Complete step-by-step instructions to build your custom 30% mechanical keyboard

Prerequisites

Time Required

2-3 hours

Skill Level

Beginner

Tools Needed

Screwdriver, Keycap puller

What You'll Need

1

MonkiiBoard 60 PCB

Hot-swappable PCB with RGB underglow support

2

Aluminum Case

CNC machined case with anodized finish

3

PCB-Mount Stabilizers

Screw-in stabilizers for larger keys

4

Mechanical Switches (61x)

Your choice of Cherry MX compatible switches

5

Keycap Set

Cherry profile keycaps (60% layout)

6

USB-C Cable

For programming and daily use

Build Steps

Step 1

Prepare Your Stabilizers

Start by preparing your stabilizers. This is crucial for smooth operation of larger keys.

  • Remove stabilizers from packaging
  • Clip the extra plastic feet if using plate-mount stabilizers
  • Apply dielectric grease to wire ends
  • Lube the stabilizer housing lightly
  • Assemble the stabilizer wire and housing
üí° Pro Tip: Don't over-lube! A thin layer is all you need for smooth operation.
Step 2

Install Stabilizers on PCB

Install the prepared stabilizers onto the PCB. This keyboard requires stabilizers for spacebar, both shifts, enter, and backspace.

  • Identify stabilizer positions on the PCB
  • Insert stabilizer housing from the top of the PCB
  • Ensure the housing clips into place
  • Flip PCB over and screw in from the bottom
  • Test stabilizer movement - should be smooth with no rattling
‚ö†Ô∏è Warning: Make sure stabilizers are properly seated before screwing in. Misaligned stabilizers can cause binding.
Step 3

Insert Switches

Thanks to the hot-swap design, installing switches is quick and easy - no soldering required!

  • Inspect each switch to ensure pins are straight
  • Align switch pins with hot-swap socket holes
  • Press down firmly until switch clicks into place
  • Repeat for all 61 switch positions
  • Double-check all switches are fully seated
‚ö†Ô∏è Warning: Bent pins can damage hot-swap sockets! Always check pins before inserting switches.
Step 4

Case Assembly

Install your completed PCB assembly into the aluminum case.

  • Optional: Add foam dampening material to case bottom
  • Carefully lower PCB into the case
  • Align mounting holes and USB-C port with case cutout
  • Insert and tighten case screws (don't overtighten)
  • Verify USB port is accessible
üí° Pro Tip: Foam can significantly reduce case ping and improve acoustics!
Step 5

Flash Firmware

Program your keyboard with QMK or VIA firmware to customize layouts and RGB effects.

  • Download QMK Toolbox from qmk.fm
  • Connect keyboard via USB-C cable
  • Put keyboard in bootloader mode (press reset button)
  • Select the MonkiiBoard 60 firmware file
  • Click "Flash" and wait for completion
  • Test all keys using a keyboard tester website
üí° Pro Tip: Use VIA for real-time key remapping without reflashing firmware!
Step 6

Install Keycaps

The final step - install your keycaps and enjoy your new keyboard!

  • Start with stabilized keys (spacebar, shifts, enter, backspace)
  • Press keycaps straight down onto switches
  • Work row by row from bottom to top
  • Ensure all keycaps are properly seated
  • Give your keyboard a test type!
üéâ Congratulations! You've successfully built your MonkiiBoard 60!

Troubleshooting

Key not registering?

Remove the switch and check if pins are bent. Straighten carefully and reinsert.

Stabilizer rattling?

Remove keycap and add more lube to the stabilizer wire ends and housing.

RGB not working?

Check firmware settings and ensure RGB is enabled in VIA or QMK configuration.
